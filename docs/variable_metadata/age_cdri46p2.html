Notes Insight46 phase 2 clinical dementia rating (CDR) - a global scale which assesses dementia severity via an interview with participant and informant. Performance is rated in six domains (memory, orientation, judgement and problem solving, community activities, home and hobbies, and personal care) and then a global CDR score is calculated by combining the scores from each domain using an establish scoring algorithm. Not measured at phase 1.
