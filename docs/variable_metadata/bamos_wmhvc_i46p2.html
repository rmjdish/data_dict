Notes Insight46 phase 1 & 2 white matter hyperintensity volumes generated using the longitudinal Bayesian Model Selection (BaMoS) pipeline. These data are intended for longitudinal analysis, and should be used to reflect WMH volume change between Phase 1 and Phase 2. For cross-sectional analysis of Phase 1 global WMH volume, use the variable lesiontot_i46p1. For cross-sectional analysis using only Phase 2 WMH volumes, use the cross-sectional Phase 2 WMHV variable. This is because WMH volumes calculated in a longitudinal framework from mid-point registered images tend to be a more conservative estimate of WMH volume, incorporating information from both time-points. If only interested in WMH volume at a single time-point, cross-sectional measurements are more appropriate. For longitudinal analysis of WMH volume change, it is recommended that only this dataset is used, rather than a combination of cross-sectional measurements of WMH volume from Phase 1 and Phase 2. For phase 1 cross-sectional measurements, see file handle I46_bamos.
