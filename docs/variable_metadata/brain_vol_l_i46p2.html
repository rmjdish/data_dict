Notes Insight46 phase 2 brain volumes measured from T1 MRI as part of the longitudinal pipeline. Note that boundary shift integral measures have been quantified for some regions (whole brain, ventricles, hippocampus) which provide more precise estimates of volume change between phase 1 and 2 compared to indirectly calculating the difference in volume between each time-point. For phase 1 brain volumes measured from T1 MRI as part of longitudinal pipeline - see library file - I46_brainVolumes_L.
