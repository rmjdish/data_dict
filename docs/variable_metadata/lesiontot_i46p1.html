Notes Insight46 phase 1 white matter hyperintensity volumes (global and regional) generated using the cross-sectional Bayesian Model Selection (BaMoS) pipeline . Note that there is also a longitudinal phase 1 white matter hyperintensity volume dataset which was generated after phase 2 data collection was complete - library file I46P2_Bamos_L. For longitudinal analysis of WMH volume change, we recommend using the I46P2_Bamos_L dataset rather than a combination of cross-sectional measurements of WMH volume from Phase 1 and Phase 2. This is because WMH volumes calculated in a longitudinal framework from mid-point registered images tend to be a more conservative estimate of WMH volume, incorporating information from both time-points.
