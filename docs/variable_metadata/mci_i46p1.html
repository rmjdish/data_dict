Notes Insight46 phase 1 mild cognitive impairment (MCI) / dementia consensus. Criteria for Mild cognitive impairment (MCI) were agreed based on published criteria. The principle of these criteria is that they require an absence of dementia AND evidence of objective cognitive impairment on one or more neuropsychological tests AND subjective cognitive concerns reported by the participant or an informant. Also collected in phase 2 - see library file I46P2_NeuroPsyc_Cognitive.
