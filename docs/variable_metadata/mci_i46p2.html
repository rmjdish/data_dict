Notes Insight46 phase 2 mild cognitive impairment. Criteria for mild cognitive impairment (MCI) were agreed based on published criteria. The principle of these criteria is that they require an absence of dementia AND evidence of objective cognitive impairment on one or more neuropsychological tests AND subjective cognitive concerns reported by the participant or an informant. It was determined using phase 2 data only. Also derived at phase 1 - see library file I46_memory.
