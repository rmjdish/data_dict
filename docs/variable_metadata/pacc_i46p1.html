Notes Insight46 phase 1 Preclinical Alzheimer Cognitive Composite (PACC) - The four components of our version of the PACC were: MMSE total score, Logical Memory delayed recall score, Digit-Symbol Substitution score and FNAME-total. The four components were converted into z-scores based on the full Insight 46 sample and the four z-scores were then averaged. This has been recalculated at phase 2 - see library file I46P2_NeuroPsyc_MMSE.
