Notes Insight46 phase 2 Preclinical Alzheimer Cognitive Composite (PACC) - The four components of our version of the PACC were: MMSE total score, Logical Memory delayed recall score, Digit-Symbol Substitution score and FNAME-total. The four components were converted into z-scores based on the performance of the full Insight 46 sample at phase 1 and the four z-scores were then averaged. This was also calculated at phase 1 - see library file I46_neuropsychology.
