Notes Insight46 phase 1 brain volumes measured from T1 MRI as part of the longitudinal pipeline. Can be used for longitudinal analysis with phase 2 brain volumes. Note that boundary shift integral measures have also been quantified for some regions (whole brain, ventricles, hippocampus) which provide more precise estimates of volume change between phase 1 and 2 compared to indirectly calculating the difference in volume between each time-point - see library file - I46P2_brainVolumes_L
